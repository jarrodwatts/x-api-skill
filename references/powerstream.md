# Powerstream

Powerstream is an advanced/enterprise streaming surface. Most users should start with:
- `references/streams.md` (Filtered Stream)
- `references/account-activity.md` (webhook-style account activity)

## Common prompts this helps with

- “I need very high-volume streaming ingestion”
  - Read the Powerstream docs below and plan for reconnect/backoff, redundancy, and operational monitoring.
