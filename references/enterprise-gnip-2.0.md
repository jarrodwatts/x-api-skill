# Enterprise Gnip 2.0

This area is for **enterprise** customers (Gnip 2.0 / firehose-style products).

## Common prompts this helps with

- “I need enterprise firehose / decahose / powertrack”
  - Start with the Enterprise overview doc below and confirm plan/entitlements.

If you’re just trying to post/search/bookmarks/DMs, you probably don’t need this section.
